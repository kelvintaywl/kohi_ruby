# Kohi

Ruby package for the [Kohi](https://github.com/kelvintaywl/k-h-) Protobuf definitions.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'kohi', '~> 0.1', :git => 'https://github.com/kelvintaywl/kohi_ruby.git'
```

And then execute:

    $ bundle install
